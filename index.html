Product Overview

Black Lab Server

Installed on an IIS server in your organization's intranet, Black Lab Server is comprised of:
  • Web services that process the query execution requests from the clients.
  • The Black Lab Administration web site which is used to set up user group permissioning, database connections, query categories, queries, and parameters.
  • The Black Lab SQL Server database that stores the information loaded via the web administration site.

Black Lab Excel Add-In

  • Communicates with the Black Lab Server via web services.
  • The Add-In provides a tree view list of all queries the user is permissioned to execute.
  • Queries may be refreshed and existing result sets may be requeried with new parameters, all through a clean ribbon bar interface.

Black Lab EXE

This command line executable provides execution of Black Lab Queries in batch scripts.
  • Simply specify the query name and optional parameters.
  • Connection info. remains hidden from the script and safely encrypted on the Black Lab Server.
  • Specify output formats of XML, CSV, Pipe and XLSX.
  • Easily generate exports without embedding sql in batch programs.

Frequently Asked Questions

Q: What versions of excel does Black Lab work with?

A: Excel 2007, 2010 and 2013


Q: Can I save a spreadsheet after retrieving data and then use that sheet again after it has been closed?

A: Black Lab enabled spreadsheets retain the query information so that they may be refreshed or requeried after they have been reopened.
